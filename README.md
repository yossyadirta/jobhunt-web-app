## About Jobhunt
Jobhunt is a career portal web application for job seekers. This application consists of a client for job seekers and a client for the content management system.

## Features
### User
- Login and Register
- CRUD Jobs
- CRUD Companies

### Admin - Content Management Systems (CMS)
- Login and Register
- CRUD Jobs
- CRUD Companies
